1. Глобальные атрибуты
Доступны для всех HTML-элементов.

Атрибут	Пример	Описание
id	<div id="header">	Уникальный идентификатор
class	<p class="text-red">	Класс для CSS/JS
style	<span style="color:red">	Встроенные стили
title	<a title="Подсказка">	Всплывающая подсказка
data-*	<div data-user="123">	Пользовательские данные
hidden	<p hidden>	Скрывает элемент
tabindex	<button tabindex="1">	Порядок фокусировки
contenteditable	<div contenteditable>	Разрешает редактирование
draggable	<img draggable="true">	Можно перетаскивать
lang	<html lang="en">	Язык содержимого
dir	<p dir="rtl">	Направление текста (ltr/rtl)
2. Атрибуты событий
Используются для JavaScript.

Атрибут	Пример (<button onclick="func()">)	Описание
onclick	Выполняет код при клике
onmouseover	Срабатывает при наведении
onkeydown	При нажатии клавиши
onload	<body onload="init()">
onsubmit	<form onsubmit="validate()">
onchange	<input onchange="update()">

3. Атрибуты форм (<input>, <form>, <select>)
Атрибут	Пример	Описание
type	<input type="text">	Тип поля (text, password, email, etc.)
name	<input name="username">	Имя для отправки на сервер
value	<input value="John">	Значение по умолчанию
placeholder	<input placeholder="Введите имя">	Подсказка в поле
required	<input required>	Обязательное поле
disabled	<button disabled>	Блокирует элемент
readonly	<input readonly>	Только для чтения
checked	<input type="checkbox" checked>	Выбранный чекбокс/радио
min/max	<input type="number" min="1" max="10">	Ограничения для чисел
pattern	<input pattern="[A-Za-z]+">	Регулярное выражение
autocomplete	<input autocomplete="off">	Отключает автозаполнение

4. Атрибуты ссылок (<a>) и изображений (<img>)
Атрибут	Пример	Описание
href	<a href="https://example.com">	URL ссылки
target	<a target="_blank">	Открыть в новой вкладке
download	<a download="file.pdf">	Принудительная загрузка
rel	<a rel="nofollow">	Отношение ссылки
src	<img src="image.jpg">	Путь к изображению
alt	<img alt="Описание">	Альтернативный текст
width/height	<img width="100">	Размеры изображения
loading	<img loading="lazy">	Ленивая загрузка

5. Атрибуты мультимедиа (<video>, <audio>)

6. Мета-атрибуты (<meta>, <link>)
Атрибут	Пример	Описание

name	<meta name="description">	Мета-описание
content	<meta content="width=device-width">	Значение метатега

7. Специальные атрибуты
Атрибут	Пример	Описание

1. Базовые селекторы
css
* {}                     /* Все элементы */
div {}                   /* Все <div> */
.class {}                /* Элементы с class="class" */
#id {}                   /* Элемент с id="id" */

2. Комбинаторы
css
div p {}                 /* Все <p> внутри <div> */
div > p {}               /* <p> - прямые потомки <div> */
div + p {}               /* <p> сразу после <div> */
div ~ p {}               /* Все <p> после <div> */

3. Атрибутные селекторы
css
[title] {}               /* Элементы с атрибутом title */
[href="..."] {}          /* Точное значение атрибута */
[class^="icon-"] {}      /* Начинается с... */
[class$="-btn"] {}       /* Заканчивается на... */
[class*="error"] {}      /* Содержит подстроку */

4. Псевдоклассы
css
:hover {}                /* При наведении */
:focus {}                /* Элемент в фокусе */
:first-child {}          /* Первый потомок */
:nth-child(2n) {}        /* Чётные элементы */
:not(.class) {}          /* Все, кроме .class */

5. Псевдоэлементы
css
::before {}              /* Добавить контент ДО */
::after {}               /* Добавить контент ПОСЛЕ */
::first-letter {}        /* Стиль первой буквы */
::selection {}           /* Выделенный текст */

6. Структурные псевдоклассы
css
:first-of-type {}        /* Первый элемент типа */
:last-of-type {}         /* Последний элемент типа */
:nth-of-type(3) {}       /* Конкретный элемент */
:only-child {}           /* Единственный потомок */
7. Формы и состояние
css
:checked {}              /* Выбранный чекбокс */
:disabled {}             /* Отключённые поля */
:valid {}                /* Корректные данные */
:placeholder-shown {}    /* Показ placeholder */
8. Комбинированные примеры
css
nav a.active:hover {}    /* Наведение на активную ссылку в 
 */
input[type="text"] {}    /* Текстовые поля */
div > :first-child {}    /* Первый потомок div */

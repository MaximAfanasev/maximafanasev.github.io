LIMIT - Ограничение количества записей

Оператор LIMIT используется для ограничения количества строк, возвращаемых запросом. Особенно полезен при работе с большими таблицами.

SQL-запрос с ограничением LIMIT 10 
SELECT
    FirstName AS [First Name],
    LastName AS [Last Name],
    Email AS [EMAIL]
FROM
    customers
ORDER BY
    LastName ASC
LIMIT 10;

Объяснение запроса

В этом запросе используется LIMIT 10 для ограничения результатов:

1. Данные из таблицы customers сортируются по фамилии

2. Псевдонимы делают заголовки столбцов понятными

3. LIMIT 10 возвращает только первые 10 записей из отсортированного результата

LIMIT: 10
Первые N записей
LIMIT 5
LIMIT с OFFSET
LIMIT 5 OFFSET 10
Синтаксис MySQL
LIMIT 10, 5
Синтаксис PostgreSQL
FETCH FIRST 5 ROWS ONLY

Примечание: LIMIT обычно используется вместе с ORDER BY для получения предсказуемых результатов. Без ORDER BY возвращаемые строки могут быть в произвольном порядке.

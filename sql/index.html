PSEUDO 
SELECT
    FirstName AS 'First Name',
    LastName AS [Last Name],
    Email AS EMAIL
    Phone CELL
FROM
    customers
сортирует столбец с псевдонимом LastName в порядке убывания *** Если условие ORDER BY отсутствует, каждый запрос будет возвращать данные в том порядке, в котором они были изначально сохранены в таблице 
ORDER BY
    LastName ASC
для сортировки по нескольким столбцам 
ORDER BY
    FirstName ASC,
    LastName DESC

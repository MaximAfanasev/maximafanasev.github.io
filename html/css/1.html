Примеры позиционирования в CSS

position: static

Это значение по умолчанию. Элемент располагается в нормальном потоке документа.

Static

Этот блок имеет position: static. Свойства top, right, bottom, left и z-index не применяются.

Примечание: Элемент с position: static всегда располагается в соответствии с нормальным потоком документа.

position: relative

Элемент располагается в нормальном потоке, но может быть смещен относительно своего исходного положения.

Relative

Этот блок имеет position: relative и смещен на 20px сверху и 30px слева от своего исходного положения.

Примечание: Исходное пространство, занимаемое элементом, остается в потоке документа.

position: absolute

Элемент удаляется из нормального потока и позиционируется относительно ближайшего позиционированного предка.

Absolute

Этот блок имеет position: absolute и позиционируется относительно своего родителя (который имеет position: relative). Он расположен внизу справа.

Примечание: Если у элемента нет позиционированного предка, он будет позиционироваться относительно body.

position: fixed

Элемент удаляется из нормального потока и позиционируется относительно окна браузера.

Fixed

Этот блок имеет position: fixed и всегда остается в правом верхнем углу окна браузера, даже при прокрутке страницы.

Примечание: Полезно для создания навигационных панелей, кнопок "Наверх" и других элементов, которые должны быть всегда видны.

position: sticky

Элемент ведет себя как relative, пока не достигнет определенной точки, после чего становится fixed.

Прокрутите вниз, чтобы увидеть поведение sticky-элемента.

Lorem ipsum dolor sit amet, consectetur adipiscing elit. ... [длинный текст для прокрутки] ...

Sticky

Продолжение контента после sticky-элемента ... [еще текст] ...

Примечание: Sticky-позиционирование работает как гибрид relative и fixed. Элемент становится "липким", когда достигает заданного положения при прокрутке.

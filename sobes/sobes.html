Списковые включения (List Comprehensions)
new_list = [expression for item in iterable if condition]
expression — вычисляется для каждого элемента
item — текущий элемент
iterable — любой объект
condition - для фильтрации элементов


numbers = [1, 2, 3, 4, 5, 6]
squares_of_evens = [n**2 for n in numbers if n % 2 == 0]
print(squares_of_evens)  # Вывод: [4, 16, 36]


Генераторы — это итераторы, которые создают последовательности значений по мере необходимости. 
Они позволяют экономить память, так как не нужно хранить все значения в памяти. 
Генераторы создаются с помощью функции, содержащей оператор yield, 
или с помощью генераторов выражений.


def my_generator():
    yield 1
    yield 2
    yield 3

gen = my_generator()
for value in gen:
    print(value)  # Вывод: 1, затем 2, затем 3

Списковые включения (List Comprehensions)
new_list = [xx for yy in list if aa]
xx — вычисляется для каждого элемента
yy — текущий элемент
list — любой объект
aa - для фильтрации элементов

numbers = [1, 2, 3, 4, 5, 6]
squares_of_evens = 
[n**2 for n in numbers if n % 2 == 0]
print(squares_of_evens)  # Вывод: [4, 16, 36]

Генераторы — это итераторы, 
которые создают последовательности 
значений по мере необходимости. 
Генераторы создаются с помощью функции, 
содержащей оператор yield, 
или с помощью генераторов выражений.


def my_generator():
    yield 1
    yield 2
    yield 3

gen = my_generator()
for value in gen:
    print(value)  # Вывод: 1, затем 2, затем 3


gen_exp = (n**2 for n in range(6) if n % 2 == 0)
for value in gen_exp:
    print(value)  # Вывод: 0, 4, 16

Замыкание — это функция, 
которая "запоминает" окружение

def outer_function(x):
    def inner_function(y):
        return x + y  
        # inner_function "замыкает" x
    return inner_function  
    # Возвращаем ссылку на inner_function

closure = outer_function(10)  # x теперь равен 10
print(closure(5))  # Вывод: 15
print(closure(20))  # Вывод: 30

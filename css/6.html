Псевдоклассы форм CSS

:checked {}

Выбирает элементы формы, которые отмечены (checkbox, radio)

input:checked + label { background: #ff6b6b; color: white; font-weight: bold; }

Radio:

:disabled {} и :enabled {}

Выбирает отключенные и доступные для взаимодействия элементы формы

input:disabled { background: #4ecdc4; color: white; opacity: 0.7; cursor: not-allowed; } input:enabled { background: #ffe66d; color: #2c3e50; font-weight: bold; }

:disabled

:enabled

:required {} и :optional {}

Выбирает обязательные и необязательные для заполнения элементы формы

input:required { background: #6a89cc; color: white; font-weight: bold; } input:optional { background: #a55eea; color: white; font-weight: bold; }

:required

:optional

:valid {} и :invalid {}

Выбирает элементы с валидным и некорректным содержимым

input:valid { background: #26de81; color: white; font-weight: bold; } input:invalid { background: #fd9644; color: white; font-weight: bold; }

:valid (попробуйте ввести корректный email)

:invalid (попробуйте ввести некорректный email)

:in-range {} и :out-of-range {}

Выбирает элементы с значением в допустимом диапазоне и вне его

input:in-range { background: #2bcbba; color: white; font-weight: bold; } input:out-of-range { background: #fc5c65; color: white; font-weight: bold; }

:in-range (диапазон 1-100)

:out-of-range (диапазон 1-100)

:placeholder-shown {}

Выбирает элементы, у которых отображается плейсхолдер

input:placeholder-shown { background: #fed330; color: #2c3e50; font-weight: bold; }

:placeholder-shown (попробуйте ввести текст)

:read-only {} и :read-write {}

Выбирает элементы только для чтения и доступные для редактирования

input:read-only { background: #a5b1c2; color: white; font-weight: bold; cursor: not-allowed; } input:read-write { background: #3867d6; color: white; font-weight: bold; }

:read-only

:read-write

::placeholder {}

Стилизует текст плейсхолдера в полях формы

input::placeholder { color: #e84393; font-weight: bold; font-style: italic; }

::placeholder (двойное двоеточие!)

Важно: ::placeholder использует двойное двоеточие (::) в отличие от других псевдоклассов, потому что это псевдоэлемент, а не псевдокласс.

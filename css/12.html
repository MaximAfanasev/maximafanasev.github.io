О свойстве box-sizing

Свойство box-sizing определяет, как рассчитываются ширина и высота элемента.

border-box: width и height включают padding и border, но не margin.

content-box: width и height относятся только к содержимому, padding и border добавляются к общим размерам.

Демонстрационная область

Содержимое элемента
#element {
  box-sizing: border-box;
  width: 200px;
  height: 150px;
  padding: 20px;
  border: 5px solid #764ba2;
}

Визуализация Box Model

Border-Box

Border
Padding
Content
width = content + padding + border

Элемент с width: 200px, padding: 20px, border: 5px

Фактическая ширина: 200px

Ширина содержимого: 200 - 40 - 10 = 150px

Content-Box

Border
Padding
Content
width = только content

Элемент с width: 200px, padding: 20px, border: 5px

Фактическая ширина: 200 + 40 + 10 = 250px

Ширина содержимого: 200px

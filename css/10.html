CSS Позиционирование

Интерактивная демонстрация свойств position: static, relative, absolute, fixed и sticky

Управление свойствами

Объяснение position:

Элемент располагается в нормальном потоке документа. Свойства top, right, bottom, left и z-index не применяются.

Пример: все элементы по умолчанию имеют position: static. Они отображаются в обычном потоке документа.

Демонстрационная область

Родительский элемент
Демо-элемент
#demo-element {
  position: static;
  top: 0px;
  right: 0px;
  bottom: 0px;
  left: 0px;
  z-index: 0;
}

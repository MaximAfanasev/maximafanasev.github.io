Overflow в CSS

Интерактивная демонстрация управления переполнением контента

Управление свойствами

О свойстве overflow

Свойство overflow определяет, как вести себя с контентом, который не помещается в элемент.

visible: контент отображается за пределами элемента (по умолчанию).

hidden: контент обрезается и скрывается.

scroll: всегда добавляются полосы прокрутки.

auto: полосы прокрутки добавляются только при необходимости.

Демонстрационная область

Контент, который не помещается в контейнер и демонстрирует работу свойства overflow
.container {
  overflow: visible;
  width: 100%;
  height: 100%;
}

.content {
  width: 120%;
  height: 120%;
}

Примеры различных значений overflow

Контент
overflow: visible
Контент
overflow: hidden
Контент
overflow: scroll
Контент
overflow: auto

Полезные советы:

💡 Используйте overflow: hidden для обрезки контента и создания эффекта скругленных углов
💡 overflow: auto предпочтительнее overflow: scroll, так как полосы прокрутки появляются только когда нужны
💡 Для управления только вертикальной или горизонтальной прокруткой используйте overflow-x и overflow-y

Используйте элементы управления для изменения свойств и наблюдения за результатом в реальном времени.

Псевдоклассы CSS - nth-child и :not()

:nth-child(2) {}

Выбирает второй дочерний элемент родителя

:nth-child(2) { background: #ff6b6b; color: white; font-weight: bold; }

Пример:

1. Первый элемент 2. Второй элемент (выделен) 3. Третий элемент 4. Четвертый элемент 5. Пятый элемент

:nth-child(odd) {}

Выбирает нечетные дочерние элементы (1, 3, 5, ...)

:nth-child(odd) { background: #4ecdc4; color: white; font-weight: bold; }

Пример (6 элементов):

1. Нечетный (выделен) 2. Четный 3. Нечетный (выделен) 4. Четный 5. Нечетный (выделен) 6. Четный

:nth-child(even) {}

Выбирает четные дочерние элементы (2, 4, 6, ...)

:nth-child(even) { background: #ffe66d; color: #2c3e50; font-weight: bold; }

Пример (6 элементов):

1. Нечетный 2. Четный (выделен) 3. Нечетный 4. Четный (выделен) 5. Нечетный 6. Четный (выделен)

:nth-child(3n+1) {}

Выбирает элементы по формуле: 1, 4, 7, 10, ...

:nth-child(3n+1) { background: #6a89cc; color: white; font-weight: bold; }
Формула 3n+1:
n = 0 → 3*0+1 = 1
n = 1 → 3*1+1 = 4
n = 2 → 3*2+1 = 7
n = 3 → 3*3+1 = 10
и так далее...

Пример (9 элементов):

1. Выделен (3*0+1) 2. 3. 4. Выделен (3*1+1) 5. 6. 7. Выделен (3*2+1) 8. 9.

:nth-last-child(2) {}

Выбирает предпоследний дочерний элемент

:nth-last-child(2) { background: #a55eea; color: white; font-weight: bold; }

Пример (5 элементов):

1. Первый 2. Второй 3. Предпоследний (выделен) 4. Последний

:nth-of-type(2) {}

Выбирает второй элемент определённого типа среди братьев

p:nth-of-type(2) { background: #fd9644; color: white; font-weight: bold; }

Пример:

Div элемент

1. Первый параграф

Span элемент

2. Второй параграф (выделен)

Другой div

3. Третий параграф

Примечание: Считаются только элементы указанного типа (p), игнорируя другие

:nth-last-of-type(2) {}

Выбирает предпоследний элемент определённого типа среди братьев

span:nth-last-of-type(2) { background: #26de81; color: white; font-weight: bold; }

Пример:

1. Первый span
Div элемент
2. Второй span

Параграф

3. Предпоследний span (выделен) 4. Последний span

:not(.ignore) {}

Выбирает все элементы, у которых нет класса 'ignore'

div:not(.ignore) { background: #fc5c65; color: white; font-weight: bold; }

Пример:

Обычный div (выделен)
Div с классом ignore
Другой обычный div (выделен)
Div с другим классом (выделен)

div:not(#banner) {}

Выбирает все div, кроме тех, у которых есть id 'banner'

div:not(#banner) { background: #fed330; color: #2c3e50; font-weight: bold; }

Пример:

Обычный div (выделен)
Div с id banner
Другой div (выделен)
Анонимный div (выделен)

li:not(:first-child) {}

Выбирает все li, кроме первого ребенка в списке

li:not(:first-child) { background: #2bcbba; color: white; font-weight: bold; }

Пример списка:

  • Первый элемент (не выделен)
  • Второй элемент (выделен)
  • Третий элемент (выделен)
  • Четвертый элемент (выделен)
  • Пятый элемент (выделен)
Примечание: Можно комбинировать :not() с другими псевдоклассами

Структурные псевдоклассы CSS

:first-child {}

Выбирает первый дочерний элемент внутри родителя

:first-child { background: #ff6b6b; color: white; font-weight: bold; }

:only-child { background: #a55eea; color: white; font-weight: bold; font-size: 1.2em; }

Примеры CSS-селекторов атрибутов

Селектор [атрибут]

Выбирает элементы с указанным атрибутом, независимо от его значения.

CSS:

a[title] { color: #e74c3c; font-weight: bold; }

HTML:

<a href="#" title="Это подсказка">Ссылка с title</a>
<a href="#">Обычная ссылка</a>

Результат:

Ссылка с title
Обычная ссылка

Селектор [атрибут="значение"]

Выбирает элементы, у которых указанный атрибут имеет точное указанное значение.

CSS:

a[href="https://example.com"] { background-color: #2ecc71; color: white; }

HTML:

<a href="https://example.com">example.com</a>
<a href="https://example.org">example.org</a>

Результат:

example.com
example.org

Селектор [атрибут^="значение"]

Выбирает элементы, у которых указанный атрибут начинается с указанного значения.

CSS:

a[href^="https"] { border-left: 3px solid #f39c12; }

HTML:

<a href="https://example.com">HTTPS ссылка</a>
<a href="http://example.com">HTTP ссылка</a>

Результат:

HTTPS ссылка
HTTP ссылка

Селектор [атрибут$="значение"]

Выбирает элементы, у которых указанный атрибут заканчивается указанным значением.

CSS:

a[href$=".pdf"] { background-image: url('pdf-icon.svg'); }

HTML:

<a href="document.pdf">PDF документ</a>
<a href="document.doc">Word документ</a>

Результат:

PDF документ
Word документ

Селектор [атрибут*="значение"]

Выбирает элементы, у которых указанный атрибут содержит указанное значение в любой части.

CSS:

a[href*="example"] { text-decoration: underline wavy #9b59b6; }

HTML:

<a href="https://example.com">example.com</a>
<a href="https://test-example.org">test-example.org</a>
<a href="https://something.org">something.org</a>

Результат:

example.com
test-example.org
something.org

Примеры с классами

CSS:

div[class^="icon-"] { /* стили для иконок */ }
div[class$="-btn"] { /* стили для кнопок */ }
div[class*="error"] { /* стили для ошибок */ }

HTML:

<div class="icon-user">U</div>
<div class="icon-settings">S</div>
<div class="submit-btn">Отправить</div>
<div class="cancel-btn">Отмена</div>
<div class="error-message">Ошибка!</div>
<div class="server-error">Ошибка сервера</div>

Результат:

U
S
Отправить
Отмена
Ошибка!
Ошибка сервера

Примеры CSS-селекторов

1. div p {}

Выбирает ВСЕ элементы <p>, которые находятся внутри <div> (любой уровень вложенности)

div p { background-color: #ffe6e6; padding: 5px; }

Параграф 1 (прямой потомок)

Параграф 2 (вложенный через span)

Параграф 3 (вложенный через другой div)

2. div > p {}

Выбирает только ПРЯМЫХ потомков <p> элемента <div> (первый уровень вложенности)

div > p { border-left: 4px solid #ff6b6b; }

Параграф 1 (прямой потомок - будет выделен)

Параграф 2 (не прямой потомок - не будет выделен)

Параграф 3 (не прямой потомок - не будет выделен)

3. div + p {}

Выбирает первый элемент <p>, который идет НЕПОСРЕДСТВЕННО после <div>

div + p { background-color: #4ecdc4; color: white; padding: 10px; }
Это div

Параграф сразу после div (будет выделен)

Второй параграф после div (не будет выделен)

Какой-то span

Еще один параграф (не будет выделен)

4. div ~ p {}

Выбирает ВСЕ элементы <p>, которые идут после <div> (не обязательно непосредственно)

div ~ p { border-bottom: 2px dashed #45b7d1; padding: 5px; }
Это div

Параграф 1 после div (будет выделен)

Какой-то span между ними

Параграф 2 после div (будет выделен)

Параграф 3 после div (будет выделен)

5. a[title] {}

Выбирает все элементы <a>, у которых есть атрибут title (независимо от его значения)

a[title] { background-color: #ffeaa7; padding: 2px 5px; }

Селекторы атрибутов CSS

a[href="https://example.com"] {}

Выбирает элементы, у которых атрибут href точно равен указанному значению

a[href="https://example.com"] { background: #ff6b6b; color: white; font-weight: bold; }

Пример ссылок:

https://example.com (выделена) https://example.com/ (не выделена) https://example.com/page (не выделена) http://example.com (не выделена) https://google.com (не выделена)
Важно: Сравнение происходит точно, включая регистр и все символы. https://example.com ≠ https://example.com/

[attr^="value"] {}

Выбирает элементы, у которых атрибут начинается с указанного значения

a[href^="https"] { background: #4ecdc4; color: white; font-weight: bold; } div[class^="icon-"] { background: #ffe66d; color: #2c3e50; font-weight: bold; }

Пример ссылок (начинаются с https):

https://example.com (выделена) https://google.com (выделена) http://example.com (не выделена) ftp://example.com (не выделена)

Пример div (класс начинается с icon-):

icon-home (выделен)
icon-user (выделен)
menu-icon (не выделен)
icon (не выделен)
Практическое применение: Иконки Font Awesome часто используют классы вида fa-*, можно выбрать все: [class^="fa-"]

[attr$="value"] {}

Выбирает элементы, у которых атрибут заканчивается указанным значением

a[href$=".pdf"] { background: #6a89cc; color: white; font-weight: bold; } div[class$="-btn"] { background: #a55eea; color: white; font-weight: bold; }

Пример ссылок (заканчиваются на .pdf):

manual.pdf (выделена) report.pdf (выделена) manual.docx (не выделена) report.pdf?download=1 (не выделена)

Пример div (класс заканчивается на -btn):

submit-btn (выделен)
cancel-btn (выделен)
btn-primary (не выделен)
button (не выделен)
Практическое применение: Стилизация всех файлов определенного типа, например PDF-документов или изображений ([href$=".jpg"])

[attr*="value"] {}

Выбирает элементы, у которых атрибут содержит указанное значение в любой части

a[href*="example"] { background: #fd9644; color: white; font-weight: bold; } div[class*="error"] { background: #26de81; color: white; font-weight: bold; }

Пример ссылок (содержат "example"):

example.com (выделена) www.example.org (выделена) test.example.com (выделена) google.com (не выделена) example-domain.com (выделена)

Пример div (класс содержит "error"):

error (выделен)
error-message (выделен)
form-error (выделен)
success (не выделен)
warning-error (выделен)
Практическое применение: Выбор всех элементов с определенным паттерном в классе, например все уведомления об ошибках или все элементы с определенным префиксом/суффиксом

Сравнительная таблица селекторов атрибутов

Типы селекторов атрибутов:

Селектор Описание Пример
[attr="value"] Точное совпадение a[target="_blank"]
[attr^="value"] Начинается с a[href^="https"]
[attr$="value"] Заканчивается на a[href$=".pdf"]
[attr*="value"] Содержит div[class*="error"]
Запомните: Эти селекторы очень полезны для стилизации элементов без добавления лишних классов, используя уже существующие атрибуты HTML.

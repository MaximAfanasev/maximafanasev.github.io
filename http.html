HTTP (HyperText Transfer Protocol) HTTP – получает страницы из интернета Его придумал Тим Бернерс-Ли IP (Internet Protocol) IP – это как телефонный номер у компьютера IPv4 – короткий номер, например: 192.192.255.255 IPv6 – длинный номер, например: 1:2:3:4:5:6:7:8 DNS (Domain Name System) DNS – это как телефонная книга: превращает www.google.com в IP-адрес Как браузер получает страницу? Ты вводишь www.google.com Браузер спрашивает у DNS (телефонной книги), какой у Google номер (IP) Браузер звонит на этот номер и просит страницу Сервер отправляет страницу (HTML, CSS, JS) Когда всё загрузилось, срабатывает onload Почта в интернете (SMTP, IMAP, POP) Это правила, как отправлять и получать письма TCP (Transmission Control Protocol) TCP – это как секретный код, чтобы данные дошли без ошибок Серверы и IP У больших компаний много серверов, и у них разные IP Бывают версии IPv4 и IPv6 (старые и новые номера) Голосовые звонки (VoIP) Это как телефон, но через интернет (например, Zoom, WhatsApp) Порты (как двери в компьютере) 80 – обычная дверь для сайтов 443 – защищённая дверь (с замком) 8080, 8443 – запасные двери Коды ответов 301, 302 – "Сайт переехал, иди сюда" 404 – "Страница потерялась" DOM (Document Object Model) Браузер строит из страницы "кубики" (DOM), чтобы показать её красиво OSI (Open Systems Interconnection) Это как слоёный пирог из правил интернета (7 уровней) Как браузер общается с сервером? GET – "Дай мне страницу" POST – "Вот тебе данные (например, логин и пароль)" HEAD – "Покажи только заголовки" URL (Uniform Resource Locator) Это полный адрес сайта, например: https://www.google.com/?q=search Заголовки (Headers) Это как записки, которые браузер и сервер передают друг другу Telnet (эмуляция протокола) Это как старый телефон, чтобы вручную поговорить с сервером Коды ответов 200 ОК успешный запрос 201 Created Post запрос? 202 Accepted обработка в процессе 204 No content в ответе нет тела сообщения 300 Мультивыбор ресурс доступен в нескольких местах 301 Moved permanently Заголовок Location даст новый URL? 302 Moved temporarily ? 304 Not modified Тело не нужно отправлять снова 400 Bad request Запросите заного 401 Unathorized Не авторизован 403 Forbindden Идентификационные данные не имеют прав 404 Not found Ресурс не найден 500 Internal server error Запрос не выполнен 501 Not implemented Сервер не распознает запрос 502 Bad gateway Сервер в роли шлюза или прокси сервера с ошибкой 503 Service unavailable Сервер не может обработать запрос Заголовки http ответов

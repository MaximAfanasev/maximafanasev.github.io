200 Ok HTTP статус 200 OK — это код состояния успешного выполнения запроса в протоколе HTTP
201 Created Запрос успешно выполнен и в результате был создан новый ресурс
202 Accepted Запрос был принят на обработку, но обработка еще не завершена
204 No content Сервер успешно обработал запрос, но не возвращает никакого содержимого
300 Multiple Choices Запрос имеет несколько возможных ответов. Пользователь или пользовательский агент должен выбрать один из них
301 Moved permanently Запрошенный ресурс был окончательно перемещен в новое место. Все будущие запросы должны использовать новый URI
302 Moved temporarily Запрошенный ресурс временно перемещен в другое место. Текущий URI может быть использован для будущих запросов
304 Not modified Используется для кэширования. Указывает, что ресурс не был изменен с момента последнего запроса
400 Bad request Сервер не может обработать запрос из-за неверного синтаксиса или некорректных данных в запросе
401 Unauthorized Требуется аутентификация для доступа к ресурсу. Запрос не был применен, так как ему не хватает действительных учетных данных
403 Forbidden Сервер понял запрос, но отказывается его авторизовать. Доступ запрещен независимо от аутентификации
404 Not found Сервер не может найти запрошенный ресурс. Наиболее распространенный код ошибки в вебе
500 Internal server error Общее сообщение об ошибке, когда сервер столкнулся с непредвиденной ситуацией, которая помешала ему выполнить запрос
501 Not implemented Сервер не поддерживает функциональность, необходимую для выполнения запроса
502 Bad gateway Сервер, действуя как шлюз или прокси, получил недопустимый ответ от вышестоящего сервера
503 Service unavailable Сервер временно не может обрабатывать запросы из-за перегрузки или технического обслуживания

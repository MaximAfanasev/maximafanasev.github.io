SEO



Базовые вещи в Bitrix
1.1 Настройка метатегов
Где находятся:

В админке: Контент → SEO → Настройка SEO-полей.

Для отдельных страниц: в режиме правки страницы → вкладка SEO.

Какие теги настраивать:

title (заголовок страницы, до 60 символов).

description (описание, до 160 символов).

keywords (устарел, но иногда используется).

h1 (заголовок на странице, должен отличаться от title).

Динамические теги:
Можно использовать макросы (например, #SITE_NAME#, #SECTION_NAME#).

1.2 Добавление контента
Создание страницы:
Контент → Сайты → Страницы → Добавить.

Использование визуального редактора:

Вставка текста, изображений, таблиц.

Важно: проверять HTML-код (часто редактор добавляет лишние теги).

Компоненты:

Для вывода динамического контента (новости, товары) используйте готовые компоненты (например, news.list).

1.3 Структура сайта
Разделы и подразделы:
Логичная вложенность (например: Главная → Услуги → SEO-продвижение).

URL-адреса:
ЧПУ (человекопонятные URL) настраиваются в Настройках SEO → ЧПУ.
Пример: /uslugi/seo-prodvizhenie/ вместо /index.php?section=123.

2. Структура информационной страницы
2.1 Основные элементы
Метатеги:

title (отображается в поисковой выдаче).

description (влияет на CTR).

Заголовки:

h1 — главный заголовок (один на странице).

h2-h6 — подзаголовки (структура должна быть иерархичной).

Контент:

Текст с ключевыми словами (естественное вхождение).

Изображения с alt и title.

Списки, таблицы для удобочитаемости.

Дополнительно:

Хлебные крошки (breadcrumbs).

Внутренние ссылки (анкоры должны быть осмысленными).

3. SEO-инструменты
3.1 Анализ и метрики
Яндекс Метрика:

Настройка целей (например, отправка формы).

Анализ поведения пользователей (карта скроллинга, кликов).

Google Analytics:

Отчеты по трафику, конверсиям.

3.2 Ключевые слова
Key Collector:

Сбор ключевых слов из Яндекс.Wordstat.

Кластеризация запросов.

Google Keyword Planner:

Планировщик ключей для контекстной рекламы.

3.3 Технический аудит
Screaming Frog:

Проверка битых ссылок, дублей страниц, метатегов.

Google Search Console:

Ошибки индексации, проверка мобильной адаптации.

4. SEO-аудит по шаблону
4.1 Этапы аудита
Техническая часть:

Индексация сайта (robots.txt, sitemap.xml).

Скорость загрузки (Google PageSpeed Insights).

Адаптивность (Mobile-Friendly Test).

Контент:

Уникальность текстов.

Оптимизация изображений.

Ссылочная структура:

Внутренние ссылки (перелинковка).

Внешние ссылки (доноры, анкоры).

Анализ конкурентов:

Serpstat, Ahrefs (ТОП-10 выдачи).

5. Смежные направления (плюсы)
5.1 Маркетинг
Контент-маркетинг:

Блог компании, email-рассылки.

SMM:

Продвижение в соцсетях (трафик на сайт).

5.2 Маркетплейсы
SEO для Wildberries/Ozon:

Оптимизация карточек товаров (ключи в названии, описании).

5.3 Контекстная реклама
Яндекс.Директ/Google Ads:

Связка с SEO (минус-слова для исключения нецелевого трафика).

5.4 Локальное SEO
Google My Business:

Оптимизация профиля компании (отзывы, фото).

Ссылки с локальных каталогов.

5.5 Репутационный маркетинг
Отзовики:

Работа с отзывами на Flamp, Yell.

Ключевые термины
Метатеги — элементы HTML, влияющие на выдачу.

Анкор — текст ссылки.

Индексация — процесс добавления страниц в поисковик.

CTR — отношение кликов к показам (влияет на ранжирование).

Для углубленного изучения:

Официальная документация Bitrix.

Курсы по SEO (например, от Netology или Skillbox).

Гайдлайны Google и Яндекса для вебмастеров.

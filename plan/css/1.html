Демонстрация :empty и :root

На этой странице представлены примеры использования CSS-псевдоклассов :empty и :root. Эти мощные инструменты позволяют создавать более гибкие и адаптивные стили для ваших веб-страниц.

Псевдокласс :empty

Селектор :empty применяет стили к элементам, которые не содержат дочерних элементов или текста.

Этот блок имеет содержимое, поэтому стили :empty не применяются.
/* CSS код для :empty */
.empty-box:empty {
    background-color: #ffebee;
    border-color: #ff6b6b;
}

.empty-box:empty::before {
    content: "Этот блок пуст!";
    color: #ff6b6b;
    font-style: italic;
}

Псевдокласс :root

Селектор :root обращается к корневому элементу документа (тегу <html>) и часто используется для определения CSS-переменных (кастомных свойств).

Текущие значения переменных:

  • Основной цвет: var(--primary-color)
  • Вторичный цвет: var(--secondary-color)
  • Акцентный цвет: var(--accent-color)
/* CSS код для :root */
:root {
    --primary-color: #4e54c8;
    --secondary-color: #8f94fb;
    --accent-color: #ff6b6b;
    --text-color: #333;
    --bg-color: #f9f9f9;
    --border-radius: 8px;
    --transition: all 0.3s ease;
    --box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.dark-theme {
    --text-color: #f0f0f0;
    --bg-color: #333;
}
